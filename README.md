# ember-route-mocks

An addon to support mocking RouterService, RouteInfo & Transition.

## Compatibility

- Ember.js v3.24 or above
- Ember CLI v3.24 or above
- Node.js v12 or above

## Installation

```
ember install ember-route-mocks
```
